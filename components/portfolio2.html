Meridian Main Website

This project is one of the biggest that I’ve worked on. It was for a Credit Union, and we wanted to make them different from the "Big 5" Canadian banks. We pitched 3 different layouts which you will see below.

The welcome section has a cinematography as the background. Users can access their accounts directly from this section, and if they want to access the menu, they can access it from the menu button at the top.

When the user starts scrolling, the navigation sticks to the top of the page, so it’s always there to help.

Meridian is a company that values their community and customers, so instead of filling the sections with a lot of information about their products or services, we went with placing community news on the homepage.

This is another concept that plays it safe. It is clean and gives a little more information about the services Meridian provides.

The third concept uses the fixed, full height, left navigation style that remains open. This style is a combination of the previous two. Because navigation is on the left, we have the full content area available for actual content.
